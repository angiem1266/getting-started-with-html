How the Web Works

The World Wide Web consists of servers that store HTML documents that are broadcasted to the internet and shown in a web browser on a computer.


HTML

HTML stands for HyperText Markup Language. HTML consists of the below:

- Text content (what you see)
- Markup (what it looks like)
- References to other documents
- Links to other pages


Tags & Elements

HTML documents consist of text documents which are built with elements that consist of tags and attributes. Some tags are considered void tags, meaning they have no end tags. Examples of this are < img > and < br >. Paragraph or < p > tags are not considered void as they need a closing tag of < /p > with the contents in between.


Inline vs Block Elements

Not all Elements are the same. Some are Inline whereas others are Block.

Block Elements have what can be called an invisible box around the tag which means they can be re-sized. They will start a new line when used. Some examples are < div > and < form >.

Inline Elements are elements that do not show as a line break and are as is said in their name, are inline with the text. Some examples are < span > and < strong >.
